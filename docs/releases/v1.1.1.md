# Focus Pet v1.1.1

发布状态：已发布

## 概要

v1.1.1 是 Focus Pet Cloud 闭环 patch release。它把 v1.1.0 之后补齐的好友码错误反馈、Cloud 在线状态刷新、桌面端 Cloud 连接状态和增强线上 smoke 验证打进正式下载包。

## 变更

- Cloud 添加好友失败时返回明确错误，桌面端不会把无效好友码显示成添加成功。
- Cloud 用户上线、下线和添加好友后，会向本人和好友广播最新 state，用于刷新在线状态。
- 桌面聊天面板显示 Cloud 连接状态：正在连接、已连接、已断开、连接异常。
- Cloud 模式下语音/视频按钮只在账号、好友和 WebSocket 都可用时启用。
- `cloud:smoke` 增强为验证无效好友码拒绝、好友上线刷新、好友离线刷新、WebSocket 信令和 Cloud StepFun 屏幕检查。
- `docs/development-plan.md` 已对齐当前 ad-hoc signed 发布路径，不再把 Apple notarization 作为本轮发布门禁。

## 签名状态

本版本按用户要求不做 Apple Developer ID 签名和 notarization。macOS 包会进行 ad-hoc 签名，但不是 Apple-notarized build。首次启动时，macOS Gatekeeper 可能要求在“系统设置”中手动允许，或通过右键“打开”启动。
