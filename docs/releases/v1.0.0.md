# Focus Pet v1.0.0

Initial GitHub release for Focus Pet.

## Highlights

- Desktop pet with transparent always-on-top Electron window.
- Task management, current-task selection, focus/study/game/distraction state detection, and daily review.
- Generated full-body pet animation set and chat GIF sharing assets.
- Optional screen monitor and LLM self-check remain opt-in.
- Local social chat, media messages, pet GIF sharing, and WebRTC signaling.
- Low-memory runtime architecture: optional social, diagnostics, screen monitor, LLM self-check, chat WebSocket, and GIF previews load on demand.

## macOS Signing

This release is ad-hoc signed because no Developer ID certificate or Apple notarization credentials were provided in the local environment. The app bundle verifies with `codesign`, but macOS Gatekeeper may still require manual approval on first launch.
