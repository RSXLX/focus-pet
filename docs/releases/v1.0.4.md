# Focus Pet v1.0.4

计划状态：开发中

## 目标

v1.0.4 用于关闭屏幕检查的默认社交副作用，让“自我检查”成为本机优先能力。

## 变更

- 手动或自动屏幕检查默认只返回到本机 UI。
- 只有 `socialActivityShareLevel=screen-summary` 时才发布屏幕摘要。
- 屏幕摘要分享不再默认保存截图媒体。
- 新用户默认关闭复盘 LLM，避免下载后仍要求本机 LLM key。
- 首次发现系统权限缺失时，引导用户进入权限设置。

## 验收

- 无本机 StepFun key 时，屏幕检查可以走 Focus Pet Cloud。
- 默认屏幕检查不会启动本地聊天服务。
- 默认屏幕检查不会保存截图到聊天媒体。
- `npm test`、`npm run check`、`npm run verify:pet-render` 通过。
