# Focus Pet v1.0.3

Maintenance release focused on publishing the full desktop pet package again and keeping the StepFun key outside the downloadable app bundle.

## Highlights

- Publishes the default public DMG/ZIP from `npm run release:mac`, which packages the full desktop pet app instead of the optional Cloud `/client` wrapper.
- Separates the optional remote chat/call client package under the `Focus Pet Client` app name and `dev.focus-pet.client` bundle ID.
- Keeps StepFun screen checks routed through Focus Pet Cloud, so downloaded apps can call the backend without embedding the provider API key.
- Updates README and system docs so public downloads point to the full desktop pet release.
- Adds regression tests for release documentation and package naming boundaries.

## Downloads

- `Focus-Pet-1.0.3-mac-arm64.dmg`
- `Focus-Pet-1.0.3-mac-arm64.zip`

## macOS Signing

This release is ad-hoc signed because no Developer ID certificate or Apple notarization credentials were provided in the local environment. The app bundle verifies with `codesign`, but macOS Gatekeeper may still require manual approval on first launch.
