# Focus Pet v1.1.3

发布状态：已发布

## 概要

v1.1.3 是 Cloud 通话验收和桌面端可用性 patch release。它把 v1.1.2 之后的桌面聊天、设置窗口、Cloud relay 验证和人工通话验收记录收敛进正式版本，避免本地能力和 GitHub Release 资产脱节。

## 变更

- 设置、引导和复盘继续使用更大的客户端面板，悬浮桌宠只保留消息、任务、设置、收起和退出等核心入口。
- 顶部聊天入口改为“消息”，减少悬浮窗文字密度。
- Cloud 通话状态行固定可见且可点击复制验收摘要。
- 通话状态会展示连接状态、远端音频/视频到达和 relay 标记，方便真实双机验收。
- 新增 `npm run cloud:webrtc:verify`，用生产 WSS 信令和 relay-only WebRTC 验证 audio/video 媒体轨道。
- 新增 `npm run call:acceptance`，把两台真实电脑各自复制的通话状态摘要合并成本地 Markdown 验收记录。
- `npm run cloud:smoke` 覆盖 Cloud 文字/图片消息持久化、好友在线/离线刷新、WebSocket 通话邀请和 `/api/screen-check`。
- 文档同步说明 Modal 只部署 Focus Pet Cloud 和下发 TURN ICE 配置，真正 TURN relay 来自托管 TURN/coturn。

## 签名状态

本版本按用户要求不做 Apple Developer ID 签名和 notarization。macOS 包会进行 ad-hoc signing，但不是 Apple-notarized build。首次启动时，macOS Gatekeeper 可能要求在“系统设置”中手动允许，或通过右键“打开”启动。

- `cloud:webrtc:verify` 是本机 relay 媒体门禁，不能替代两台真实电脑、不同网络下的人工语音/视频验收。
- 这不是后台静默自更新。当前未做 Apple notarization，不适合在后台直接替换正在运行的 App。
