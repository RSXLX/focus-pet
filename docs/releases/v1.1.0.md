# Focus Pet v1.1.0

发布状态：准备发布

## 概要

v1.1.0 把 Focus Pet Cloud 账号、好友码、后端屏幕检查代理和一对一语音/视频接入完整桌面宠物。普通用户下载完整桌宠后，可以在应用内创建自己的 ID、查看好友码、添加好友，并通过 Cloud WebSocket 建立 WebRTC 通话。

## 变更

- 桌面聊天面板新增 Cloud 账号区。
- 用户可以在完整桌宠里创建自己的 ID。
- 用户可以查看自己的好友码。
- 用户可以输入好友码添加对方。
- 桌面端通过 Cloud WebSocket 进行 WebRTC 信令。
- 现有 `RTCPeerConnection` 语音/视频逻辑复用，不重写通话栈。
- Focus Pet Cloud 后端代理 StepFun 屏幕检查，桌面端不需要内置 StepFun key。
- Focus Pet Cloud 已配置 Metered TURN，跨网络语音/视频连接更稳定。
- 发布门禁新增 `cloud-health` 和 `cloud:turn:verify`。
- Cloud 模式下文字、文件和宠物动图暂不走 Cloud，避免临时新增一套消息存储。

## 签名状态

本版本按用户要求不做 Apple Developer ID 签名和 notarization。macOS 包会进行 ad-hoc 签名，但不是 Apple-notarized build。首次启动时，macOS Gatekeeper 可能要求在“系统设置”中手动允许，或通过右键“打开”启动。

## 验证

- `npm run cloud:turn:verify` 通过。
- `npm run cloud:smoke -- --skip-screen-check` 通过。

## 验收

- 新装 App 打开聊天页可以创建 ID。
- 创建后显示好友码。
- 两台设备可以通过好友码互加。
- 好友在线状态可以刷新。
- 语音通话可以建立。
- 视频通话可以建立。
- 重启 App 后 ID 不丢。
