# Focus Pet v1.0.1

Maintenance release focused on update notifications.

## Highlights

- Adds GitHub Release based update checks.
- Enables automatic update checks by default for app maintenance.
- Shows a system notification when a newer release is available.
- Clicking the notification or manually checking updates opens the GitHub Release download page.
- Falls back to the public `releases/latest` redirect when the GitHub API is rate-limited.
- Keeps installation user-driven: the app does not silently download, replace, or restart itself.

## macOS Signing

This release is ad-hoc signed because no Developer ID certificate or Apple notarization credentials were provided in the local environment. The app bundle verifies with `codesign`, but macOS Gatekeeper may still require manual approval on first launch.

## Verification

- `npm run check`
- `npm test`
- `node scripts/release-preflight.js --check error-log`
- `node -e "const { checkLatestVersion, DEFAULT_UPDATE_FEED_URL } = require('./src/update-service'); checkLatestVersion({ currentVersion: require('./package.json').version, feedUrl: DEFAULT_UPDATE_FEED_URL }).then(result => console.log(result))"`
- `npm run release:mac:controlled`
