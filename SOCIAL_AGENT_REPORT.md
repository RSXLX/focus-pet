# Focus Pet 社交化桌面宠物 Agent 报告

## 已调研资料

- Electron `BrowserWindow` / Window Customization：适合做透明、无边框、置顶、动态缩放的桌面宠物窗口。
- Electron `Notification`：适合系统通知，但 macOS 未签名应用可能通知失败，因此本项目优先使用宠物气泡和弹出聊天框。
- MDN WebSocket API：支持两端低延迟双向通信，适合桌面端和浏览器第二端实时聊天。
- MDN `getUserMedia`：获取麦克风/摄像头需要用户授权，适合语音聊天和视频能力入口。
- MDN `MediaRecorder`：可把麦克风音频录成 `webm`，适合语音消息。
- MDN File API：支持选择本地图片/视频并在聊天中发送和预览。

## 当前实现

### 桌面宠物 Agent

- 透明无边框置顶窗口。
- 默认小体积、不大面积挡鼠标。
- 默认系统级 click-through，鼠标进入宠物/气泡/面板时才接管点击。
- 支持按住宠物本体拖动，轻点宠物展开/收起。
- 参考旧 QQ 宠物交互：右键宠物打开照料菜单，支持喂食、清洁、学习、打工、玩耍、休息。
- 有心情、精力、亲密度三项轻量状态反馈。
- 不同照料动作会触发短动画和气泡反馈。
- 闲置一段时间后，宠物会低频主动提醒继续任务。
- 每分钟读取前台 App 和窗口标题，判断 `work`、`distracted`、`unknown`、`permission`。
- 疑似分心/未知/权限问题时，触发智能弹出聊天框，但有冷却时间，避免一直打扰。

### 提醒功能

- 轻提醒：宠物气泡显示当前状态。
- 任务提醒：可编辑今日任务，保存到 `~/.hermes/focus-watchdog/today_tasks.md`。
- 复盘提醒：统计近 24 小时工作/分心/未知时间和常用 App。
- 智能弹出：不是常驻聊天框，只有点击聊天、收到消息、疑似偏离任务时弹出。

### 社交聊天

- Electron 内置本地 HTTP + WebSocket 服务。
- 默认端口：`47321`。
- 桌面端和浏览器第二端可互相通信。
- 第二端地址：`http://127.0.0.1:47321/client`。
- 支持好友列表、加好友、文本消息。
- 支持图片、视频文件发送和预览。
- 支持语音消息录制和发送。
- 消息与媒体保存在 `~/.hermes/focus-watchdog/social/`。

## 桌面宠物 Agent 继续优化方向

### 交互体验

- 做多种宠物情绪：专注、开心、困惑、催促、休息、庆祝。
- 让宠物有“低打扰模式”：只在高置信度分心时弹出。
- 引入可配置提醒语气：温柔、严格、伙伴、教练。
- 加入番茄钟状态：专注 25 分钟、休息 5 分钟、长休息。
- 给任务完成添加庆祝动画，强化正反馈。

### 智能判断

- 当前只靠 App 和窗口标题，隐私好但上下文弱。
- 可选增强：浏览器域名白/黑名单、项目关键词、日程/待办同步。
- 可选增强：只在本地做 OCR 摘要，不上传截图。
- 引入“当前目标”概念：用户每天设定 1–3 个目标，宠物按目标判断偏离。
- 引入学习曲线：根据用户习惯自动调整提醒频率和阈值。

### 社交属性

- 加入“学习搭子”房间：两端互相监督在线状态。
- 加入共同番茄钟：双方同时开始/结束专注。
- 加入轻量打卡：完成一项任务后同步给好友。
- 加入互相鼓励/提醒模板，降低聊天成本。
- 加入排行榜或连续专注天数，但要避免过度游戏化焦虑。

### Agent 能力

- 宠物不只是提醒，而是能帮你拆任务。
- 当你偏离时，弹出“回到任务的下一步”按钮。
- 和 Hermes/本地 Agent 联动：一键生成学习计划、复盘、资料总结。
- 根据聊天内容自动提取待办。
- 根据每日复盘自动调整明日任务建议。

### 安全与隐私

- 明确区分本地数据和联网数据。
- 媒体文件默认只保存在本机。
- 如果未来支持公网好友，需要加认证、权限、限流、加密。
- 语音/视频默认不自动录制，只能用户主动点击。
- 提供一键清空历史和媒体缓存。

## 推荐产品路线

### V1：本地桌面宠物

- 稳定透明置顶和 click-through。
- 任务/复盘/轻提醒。
- 本地第二端 WebSocket 聊天。
- 图片/视频/语音消息。

### V2：学习搭子社交

- 好友邀请码。
- 在线状态和共同番茄钟。
- 未读消息和提醒节流。
- 局域网或云端中继。

### V3：Agent 教练

- 每日目标设定。
- 偏离时给出下一步建议。
- 自动复盘和明日计划。
- 语音对话和语音转文字。

### V4：电商运营工作台联动

- 把宠物提醒和你的电商自动化系统连接。
- 今日任务来自电商 Agent 的看板。
- 宠物提醒你推进选品、内容、上架、客服、交付等关键节点。
- 社交端可以让合作者/监督者看到进度摘要，而不是隐私细节。
